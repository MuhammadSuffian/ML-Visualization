# Visualize  Machine Learning 

A minimal example of visualizing a Linear Regression model using Plotly.

## What it does

- Loads the `Advertising.csv` dataset
- Fits a Linear Regression model (TV spend → Sales)
- Plots actual data points and the regression line using Plotly

## Requirements

```
pandas
numpy
scikit-learn
plotly
```

## Usage

```bash
pip install pandas numpy scikit-learn plotly
```

Open `Visualize a ML.ipynb` and run all cells.
